# Available Tools

## Filesystem
- `read_file` - Read file contents
- `write_file` - Create or overwrite files
- `edit_file` - Replace exact text in files
- `list_dir` - List directory contents

## Shell
- `execute` - Run shell commands (subject to safety blocklist)

## HTTP
- `http_request` - Make HTTP API calls (GET, POST, PUT, DELETE, PATCH)

## Memory
- `save_memory` - Save important information to daily notes
- `search_memory` - Search through past memory notes

## Sub-agents
- `spawn_subagent` - Create an isolated sub-agent for specific tasks

Use these tools to accomplish your goals efficiently and safely.
